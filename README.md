### Para hacer funcionar firebase hay que Configura las siguientes variables

- VITE_APIKEY=
- VITE_AUTH_DOMAIN=
- VITE_PROJECT=
- VITE_STORAGE=
- VITE_MESSAGIN=
- VITE_APP_ID=
